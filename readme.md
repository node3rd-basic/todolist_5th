# nodejs 3기 Basic Class

## 목표
- 기본적인 웹 통신, 웹 서비스에 대한 흐름을 익힌다.
- 웹 서비스에서 가장 기본이 되는 게시판 형태를 작성 할 수 있다.
- git의 기본 사용법을 알 수 있다.
- 회원가입, 인증에 대한 개념을 익힌다.
- 관계형 DB (mysql) 에 대해서 기본적인 기능을 익힌다.
- Todolist 프로그램을 작성 할 수 있다.
